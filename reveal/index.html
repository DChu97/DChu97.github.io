How the MPC came about

Before the MPC was invented, animators found it difficult to create tracking shots using the traditional animation style. Animating forward motions became costly and time consuming. The MPC fixed this by creating a realistic sense of depth in a cartoon setting.

  • The MPC has been used for about 56 years
  • 101 Dalmatians, Pinocchio, Dumbo, Bambi, etc. were all made using the MPC
  • "Snow White and the Seven Dwarfs" was the first and "The Little Mermaid" was the last movie made using this technology
